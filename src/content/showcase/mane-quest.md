## Overview

*Mane Quest* is the studio's flagship pocket roguelike. Each run drops you onto a procedurally generated savanna where weather, prey behavior, and rival prides all evolve hour by hour. Runs typically last 8–12 minutes, making it a comfortable fit for a commute or coffee break.

## Highlights

- **One-thumb control scheme.** Designed mobile-first; every action is a tap or swipe.
- **Deterministic seeds.** Share a seed code with friends and race the same map.
- **Offline by default.** No login, no ads. Scores sync when you reconnect.

## Tech

Built with TypeScript and a custom canvas renderer. Web and Android share the same core, with platform-specific input layers.
